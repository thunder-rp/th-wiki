# Débuter l'aventure

Bienvenue dans la ville de Thunder RP.&#x20;

Afin de bien débuter votre aventure, voici une liste d'étapes essentielles à votre intégration dans notre belle ville !&#x20;

## Un long voyage…

Après un long voyage en avion, vous sortez de l'aéroport avec vos nombreux bagages et quelques sous en poche.

Si vous ne connaissez personne en ville, Pédro, le vendeur de vélos vous attendra devant celui-ci. Vous avez donc la possibilité de **louer un vélo** pour vous déplacer plus facilement vers ville.&#x20;

## Vos papiers

A l'aéroport, en regardant dans votre sac vous vous rendez-compte que vos papiers ont été égarés pendant le voyage !

Il vous faut donc vous rendre **au gouvernement** afin de **refaire votre carte d'identité** pour la modique somme de **500$**.

<figure><img src="https://cdn.thunder-rp.fr/p/files/thunderp/wiki/debuter_laventure/carte.png" alt=""><figcaption><p>La secrétaire du Gouverneur, Jessica, qui vous attend pour vos papiers</p></figcaption></figure>

## Votre permis

Etant vraiment malchanceux, vous avez également égaré votre **permis de conduire.**

Il faudra vous rendre **à l'auto-école**, indiquée sur votre carte, pour le repasser.

<figure><img src="https://cdn.thunder-rp.fr/p/files/thunderp/wiki/debuter_laventure/permis.png" alt=""><figcaption><p>L'auto-école</p></figcaption></figure>

## Pôle emploi

Lorsque l'on veut s'acheter un véhicule, un logement, des denrées, un téléphone, une ressource est nécessaire : **l'argent**.

Différents métiers sont disponibles afin de se **faire de l'argent facilement** à conditions que vous mettiez **1000$** de côté pour vous acheter du matériel.

## Les oranges

Si vous avez la main verte, vous avez la possibilité de **récolter des oranges** dans les vergers de Paleto Bay.&#x20;

Deux possibilités s'offrent donc à vous :&#x20;

* Les vendre dans des magasins
* Les acheminer vers une installation de transformation pour en faire du jus d'orange et pour les vendre encore plus cher.

<figure><img src="https://cdn.thunder-rp.fr/p/files/thunderp/wiki/debuter_laventure/oranges.png" alt=""><figcaption><p>Verger d'orangers</p></figcaption></figure>

## La pêche

Vous avez la possibilité d'acquérir une canne à pêche dans une supérette pour partir à la pêche aux poissons !

Bien que cela puisse ne pas être une tâche aisée, les récompenses potentielles sont considérables.

N'hésitez donc pas à vous lancer dans cette aventure aquatique !&#x20;

<figure><img src="https://cdn.thunder-rp.fr/p/files/thunderp/wiki/debuter_laventure/peche.png" alt=""><figcaption><p>Tonton qui pêche</p></figcaption></figure>

## La mine

Si vous adorez passer votre temps à charbonner, vous avez pour finir la possibilité de vous acheter une pioche dans une supérette et d'aller miner !

Vous obtiendrez différents métaux et ressources que vous pourrez vendre au Pawnshop pour vous faire un peu d'argent !&#x20;
